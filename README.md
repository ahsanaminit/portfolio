# Portfolio

This repository contains a Vite + React starter scaffold with Tailwind CSS and Framer Motion for animations.

Getting started

1. Install dependencies

   npm install

2. Run dev server

   npm run dev

3. Build for production

   npm run build

Notes

- The hero uses Framer Motion for simple, performant animations. Add Lottie or GSAP if you need more advanced illustrations/scroll timelines.
- To deploy: I recommend Vercel for zero-config React deployments. You can also add a GitHub Actions workflow to publish the built `dist/` to GitHub Pages.
